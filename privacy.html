Privacy Policy

Effective date: 2026-05-30 · Last updated: 2026-05-30

1. Who we are

LabGames is an automated gaming-content pipeline operated by an individual content creator based in Brazil. It generates and publishes original videos about video games on the Facebook page @LABGAMES and on TikTok using the same handle. For questions about this policy, contact suporte@spectralsolutions.com.br.

2. Scope

This policy describes how the LabGames pipeline handles data when it interacts with the TikTok and Facebook platforms via their official APIs. The pipeline is operated by, and for, a single content creator and does not collect or process personal data of any third party or end user.

3. Data we access

To publish videos on the operator's own social-media accounts, the pipeline accesses:

  • TikTok OAuth tokens (access token and refresh token) generated when the operator authorizes the application. These are stored locally on the operator's machine (encrypted environment file) and are used solely to call the TikTok Content Posting API.
  • Basic TikTok account information (open id, display name) returned by the TikTok OAuth response, used only to confirm the authorization succeeded.
  • Facebook page access tokens and page identifiers, used to upload videos to the operator's own Facebook page via the Meta Graph API.
  • Public game-data and images retrieved from third-party APIs (RAWG.io) and public RSS feeds of gaming news outlets. No personal data is fetched from these sources.

4. Data we do not collect

  • We do not collect personal data of TikTok or Facebook viewers.
  • We do not track, profile, or sell any user information.
  • We do not place cookies or analytics on this site.

5. How data is used

The OAuth tokens are used exclusively to authenticate calls to the TikTok and Facebook APIs in order to upload the operator's own pre-rendered video files and the associated caption text. Tokens are never shared with any third party and are never transmitted outside of the official TikTok and Facebook API endpoints.

6. Storage and security

Tokens are stored in a local environment file (.env) on the operator's workstation, excluded from any version-control system. The file is accessible only to the operator's local user account. Access tokens follow the expiration policies of each platform (TikTok access token ~24 hours; refresh token up to 365 days). When a token expires, the pipeline either refreshes it via the platform's official refresh endpoint or requires re-authorization by the operator.

7. Data sharing

We do not share data with any third parties. The only network destinations the pipeline contacts are:

  • TikTok official API (open.tiktokapis.com)
  • Meta Graph API (graph.facebook.com, graph-video.facebook.com)
  • RAWG.io public games database
  • Public RSS feeds of gaming news outlets
  • ElevenLabs text-to-speech API (for narration generation)
  • Anthropic Claude (for editorial generation)

8. Retention

Authentication tokens are retained on the operator's machine for as long as they are valid. Generated videos and intermediate artifacts (transcripts, audio files) are retained locally and are not transmitted anywhere except the social platforms where they are published. The operator may delete any local artifact at any time.

9. Your rights

Because this application processes only the operator's own account data, no third-party user data is collected. The operator may revoke the application's access to TikTok or Facebook at any time via the respective platform's user settings, which immediately terminates the pipeline's ability to publish on their behalf.

Under Brazil's Lei Geral de Proteção de Dados (LGPD, Law 13.709/2018) and the EU's GDPR, individuals retain the right to access, correct, or delete personal data held by a data controller. Requests can be sent to suporte@spectralsolutions.com.br.

10. Children's privacy

LabGames is not directed at children under 13. The platforms on which it publishes (TikTok, Facebook) enforce their own age requirements for account ownership.

11. Changes to this policy

Material changes to this policy will be reflected here with an updated effective date. Continued use of the pipeline after such changes constitutes acceptance of the new policy.
